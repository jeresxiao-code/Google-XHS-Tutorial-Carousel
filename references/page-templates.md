# Page Templates

## Shared Style Prompt

Use this at the top of a multi-page prompt set:

```text
Create a 3:4 vertical Xiaohongshu educational carousel card in a clean Google-inspired tutorial style. White and very pale blue-gray background, Google blue as the main accent, small red/yellow/green accents, deep navy oversized Chinese headings, white rounded information cards with soft shadows, simple colored line icons, search-bar UI modules, green checkmarks, subtle corner decorations including a blue ring at top left, pale dot grid at top right, yellow quarter circle bottom left, green arc bottom right, and a centered bottom page-number pill. Mobile-first, crisp, organized, collectible tutorial-card feeling.
```

## Cover Page

Use for page 01.

```text
Page type: cover.
Top area: Google-style logo/search theme or rounded search bar.
Main title: "<exact title>", split into 2 large lines; first line deep navy, second line Google blue.
Subtitle: "<short subtitle>" under the title, centered.
Bottom/mid tags: 3 small rounded feature cards with line icons and short labels.
Footer: page number pill "01/<total>".
Avoid: dense text, tiny labels, clutter, official endorsement language.
```

Best for titles such as:

- 外贸新人如何 / 开发海外客户
- 新人开发海外客户 / 按这个 SOP 做
- 3 个实战案例 / 照着搜就行

## Logic Overview Page

Use for explaining the mental model before steps.

```text
Page type: logic overview.
Header label: "<series label>".
Main title: "<big title>" with one emphasized blue keyword.
Subline: a short process sentence with arrows.
Body: 4 vertical cards in one row or 2x2 grid. Each card has a number badge, large icon, bold heading, and 1 short explanation line.
Bottom tip strip: pale yellow with one strong takeaway.
Footer: page number pill.
```

## Command Grid Page

Use for 4-6 search commands or tool tips.

```text
Page type: command grid.
Main title: "<title>".
Body: 2-column grid of 4 or 6 white cards.
Each card: blue number badge, bold command name, short Chinese explanation, rounded search/query pill with exact example text.
Bottom CTA/tip strip: blue or pale blue with one sentence.
Footer: page number pill.
```

## Combination Search Page

Use for 3 large examples.

```text
Page type: combination search.
Main title: "<title>".
Body: 3 stacked wide cards.
Each card: number badge at left, bold formula title, search bar with exact query, green check explanation line.
Optional bottom note: synonym list or replacement rule in a pale strip.
Footer: page number pill.
```

## Lookup Table Page

Use for domain suffixes, country lists, company suffixes, role words, file types.

```text
Page type: lookup table.
Main title: "<title>".
Body: two white rounded panels side by side.
Each panel: colored icon, panel heading, 5-7 compact rows.
Rows: small icon or colored pill on left, text on right.
Bottom: direct-use search examples in rounded pills.
Takeaway strip: green check + one sentence.
Footer: page number pill.
```

## Source Category Page

Use for 4 categories of websites, channels, tools, or resources.

```text
Page type: source categories.
Main title: "<title>".
Body: 2x2 grid cards.
Each card: number badge, category heading, large line icon, search formula, short checkmarked use case.
Bottom checklist panel: what to screen before using the source.
Footer: page number pill.
```

## Contact Finding Page

Use for email/contact/person-finding workflows.

```text
Page type: contact finding.
Main title: "<title>".
Body: 4 stacked horizontal cards.
Each card: blue number circle, icon on left, bold method title, short explanation or example pills.
Bottom checklist: at least 5 fields to record.
Final tip strip: orange/yellow with one behavioral warning.
Footer: page number pill.
```

## Case Study Page

Use for copyable examples.

```text
Page type: case studies.
Main title: "<title>".
Body: 3 stacked case cards.
Each card: colored round icon, "案例 <n> | <case title>", search bar with exact query, green check line explaining what to inspect.
Bottom note: gray or pale blue tip strip.
Footer: page number pill.
```

## Final SOP Page

Use for the last page.

```text
Page type: final SOP.
Main title: "<title>".
Body: 6-step flow in 2 rows of square cards with arrows between steps.
Each step: blue number badge, large icon, short label.
Pitfall box: warning icon and 4-5 common mistakes in two columns.
Final save strip: yellow star icon + strong collection/save sentence.
Footer: page number pill.
```
