# Visual System

## Canvas

- Ratio: 3:4 vertical.
- Suggested size: 1080 x 1440 px or equivalent.
- Safe margin: 64-88 px.
- Background: white to very pale blue-gray, such as `#F7FAFF`, with subtle depth.

## Color

Use a Google-inspired palette:

- Main blue: `#4285F4`
- Deep title navy: `#071B3A`
- Red accent: `#EA4335`
- Yellow accent: `#FBBC05`
- Green accent: `#34A853`
- Soft gray text: `#5F6368`
- Pale card border: `#E8EEF7`
- Pale blue fill: `#EEF5FF`
- Pale green fill: `#EAF8F0`
- Pale yellow fill: `#FFF6D8`
- Pale red fill: `#FDEDEC`

Blue should dominate. Red/yellow/green should act as semantic accents: warning, highlight, success, category.

## Typography

- Main title: extra-bold Chinese sans serif, very large, deep navy plus blue emphasis.
- Secondary title: medium-bold Chinese sans serif, dark gray or navy.
- Body: clean sans serif, medium weight, gray-black.
- English examples: rounded sans serif or monospace-like UI text inside search bars.
- Keep letter spacing natural. Do not use thin fonts.

Title pattern:

```text
Line 1: deep navy
Line 2: Google blue emphasis
Optional number: oversized blue
```

## Cards

Use white rounded rectangles with soft shadows:

- Corner radius: 20-32 px.
- Shadow: soft, low opacity, downward blur.
- Border: very light blue-gray.
- Internal spacing: generous and consistent.

Common modules:

- Search bar: rounded pill, left search icon, gray border, English query text.
- Number badge: blue circle with white number.
- Icon circle: pale color background with line icon.
- Tip bar: pale yellow or pale blue horizontal rounded strip.
- Checklist: green check icons, short labels.
- Page number: bottom centered pill, e.g. `05/10`.

## Decorations

Keep decorative elements soft and fixed:

- Top-left partial blue ring.
- Top-right pale blue dot grid.
- Right-side translucent pale blue arc.
- Bottom-left pale yellow quarter circle.
- Bottom-right pale green arc.
- Tiny colored separator lines around the series label.

Decorations should be background rhythm, never primary content.

## Icon Style

- Simple line icons, 2-3 px stroke feel.
- Use search, globe, mail, document, clipboard, table, storefront, group, calendar, map pin, alert, star.
- Each icon uses one Google accent color and a pale circular glow behind it.
- Keep icon sizes consistent across the page.

## Brand Note

This style can reference Google-like colors and search UI patterns for tutorials. Do not claim the design is official Google material unless the user has rights and asks for official branding.
